### Thème : données bancaires et entreprise
CRUD sur les entreprises
Create POST : /api/entreprise
Read GET : /api/entreprise/:id
Update PUT : /api//entreprise/:id
Delete DELETE : /api//entreprise/:id

CRUD sur les comptes bancaires
Create POST : /api/bankAccount
Read GET : /api/bankAccount/:id
Update PUT : /api/bankAccount/:id
Delete DELETE : /api/bankAccount/:id


Entités :
- Entreprise
- Banque
- Compte bancaire
